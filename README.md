# Introduction 
Pega Financial Lines Regression Test Suite. 

# Getting Started
1.Install Katalon Studio 9.6(or check for latest Version)
2. Should have Java 17.
	 

# Build and Test
Take clone for the project from https://hccins.visualstudio.com/Product%20Team%20-%20Financial%20Lines/_git/FinancialLines_Katalon_Automation
 
# Contribute
Test cases are present in Test Cases folder to run any Test Case in your local.
Test Suites are present in Test Suites folder to run all test cases in one go.
Object Repository is maintained in separate folder with same name.
Data files are maintained in separate folder to read data from Excel.
Profiles section is maintained for Global variables.
All re-usable functions will be present in Keywords folder.
We have one WebListeners to pre and post test case/suite actions
 - Pre-actions contains opening browser, getting test caseId, setting downloads folder.
 - Post-actions contains closing browser.# Revamped_Staging_Pega_FL_Cloud
